**2Dエンドレスランナーゲーム開発**

研究室の夏課題「Google Chromeの恐竜ゲームを模したゲーム開発」として制作しました。
西部劇（Western）をテーマにしたドット絵素材を選定し、独自のBGMやシーン遷移を実装してゲーム性を高めています。

## 📂 ディレクトリ構成 / Directory Structure
このディレクトリはUnityプロジェクトのルートフォルダです。

- **`Assets/`**: スクリプト(`Scripts`), スプライト(`Sprites`), 音声(`Audio`), シーン(`Scenes`)などのゲームリソース。
- **`Packages/`**: Unityパッケージのマニフェストファイル。
- **`ProjectSettings/`**: プロジェクト設定ファイル群。
